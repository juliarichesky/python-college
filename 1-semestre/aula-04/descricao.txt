expressões booleanas
considere a = 5, b = 2 e c = 7; julgue as seguintes expressões em
Python:
▶ a < b
▶ a − c >= b
▶ 3 ∗ b < c
▶ c − b! = a
▶ c == a + b
▶ c % b < a // b
▶ b + c >= 2 ∗ a
▶ b − c == a
